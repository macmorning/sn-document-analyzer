# Product Overview

Document Analyzer is a ServiceNow application that analyzes photos and PDFs for fraud detection in HR case attachments. The system performs automated analysis on uploaded images and documents to detect anomalies, metadata inconsistencies, pixelation issues, and other indicators of potential fraud or manipulation.

## Key Features

- Automated photo and PDF analysis for HR case attachments
- Fraud risk assessment with confidence scoring (low/medium/high/critical)
- Metadata anomaly detection
- Pixelation and compression artifact analysis
- Timestamp inconsistency detection
- Role-based access control (analysts, investigators, administrators)
- Test case framework for validating analysis capabilities
